# Chrome Time Machine Plugin

## 项目简介

本插件用于记录用户的浏览历史，并通过时间管理功能展现过去的浏览状态。通过配置，可对部分不希望记录的特定网站的历史进行过滤。

## 配置说明

- 在 `config.js` 中的 `whitelist` 数组里添加不需要记录的目标网站域名，其对应的浏览历史将不会被保存。
- 如果部署在云服务或外网环境，请确保修改：
  - `manifest.json` 中的 `host_permissions` 配置，保证插件权限正确。
  - `config.js` 中的 `serverUrl`，确保数据请求发送到正确的服务器。

## 使用指南

1. 根据需求调整配置文件。
2. 安装并加载插件到 Chrome 浏览器。
3. 浏览网站，插件会自动管理并记录浏览历史（除白名单中的网站）。

请在修改配置前仔细阅读以上说明，确保插件按照预期运行。
